股票占比小数格式测试

历史表现可视化 - 图例格式

修改前格式：

AAPL (25%)
MSFT (20%)
GOOGL (15%)

修改后格式（保留1位小数）：

AAPL (25.0%)
MSFT (20.0%)
GOOGL (15.0%)

实际场景示例：

AAPL (7.5%)

✅ 修改完成

历史表现可视化图表中的股票占比现在将显示为1位小数格式，提供更精确的信息显示。

📊 技术实现

使用 JavaScript 的 toFixed(1) 方法将数字格式化为1位小数。 

name={`${stock.symbol} (${stock.allocation.toFixed(1)}%)`}
